## 🎯 درباره پروژه

یک نمونه‌کار (Showcase) از طراحی **سایت خبری / مجله‌ای** که اخبار حوزه‌های سیاسی، ورزشی، اقتصادی و... را در قالب رابط کاربری مدرن نمایش می‌دهد. هدف اصلی پروژه، تمرین و معرفی مهارت‌های توسعه‌ی Front-End است.

> ⚡ **نکته:** داده‌های نمایش داده‌شده نمونه و ثابت هستند. این پروژه صرفاً تمرینی Front-End بوده و به هیچ دیتابیسی متصل نیست.
